Python [Ab]Users

This group was started during the fall 2015 semester. It consists of faculty, staff, students, and friends of the physics departments at Metropolitain State University of Denver and University of Colorado Denver. The goal is to create and foster a collaborative environment of people interested in Python and scientific computing. [Ab]Users of all skill levels are welcome! We meet on select Fridays at 2:30 PM in North Classroom 3610 on the Auraria Campus in Denver, Colorado. A calendar of events can be found here.

A Student's Guide to Python for Physical Modeling

We use this text as our guide to learning Python. There is a paper-back copy available in NC3610 for reference. eBook versions are available for purchase from various vendors. Note: The kindle version is not compatible with Kindle Cloud Reader. You must download the Kindle app to read the book (and a Linux Kindle app does not exist). This is not the case for the Google Books version. Code samples, data sets, updates, errata, and more are available here.

Lectures on scientific computing with Python

A series of lectures on scientific computing using python. The lectures are written as Jupyter notebooks, which means that you can interactively edit and run the code (either on your personal computer with Jupyter installed, or here). Download the notebooks from the scientific-python-lectures GitHub repository (ZIP), or browse them online (read-only) using the these links:

  • Lecture-0 Scientific Computing with Python
  • Lecture-1 Introduction to Python Programming
  • Lecture-2 Numpy - multidimensional data arrays
  • Lecture-3 Scipy - Library of scientific algorithms
  • Lecture-4 Matplotlib - 2D and 3D plotting
  • Lecture-5 Sympy - Symbolic algebra
  • Lecture-6A Fortran and C integration
  • Lecture-6B HPC
  • Lecture-7 Revision Control Software

Git

Revision control is an essential aspect in software development. Git is the revision control system that we chose, as it is widely used and online hosting of our projects is free. We chose GitLab for collaboration and sharing our projects in this group. Sign up for a free account, and join the [Ab]Users! GitHub is also a great resource for your projects outside of the group! Here are a few links to get you started using Git.

  • Github for Academics: A short slidedeck giving an overview of the functionality of GitHub, .
